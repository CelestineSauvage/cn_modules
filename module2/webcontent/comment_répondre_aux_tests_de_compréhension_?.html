Les tests de compréhension sont identifiables par ce pictogramme :

Les tests de compréhension vous permettent de vous autoévaluer après avoir visionné chaque vidéo.
Ils sont de deux types :

1. Des "tests" qui sont :

  • soit des quizz comportant des questions à choix unique ou à choix multiple ;
  • et/ou des questions ouvertes qui demandent réflexion, recherches, analyses et que nous vous conseillons de rédiger d'abord dans un logiciel de traitement de texte, pour le copier/coller ensuite dans le champs texte du questionnaire prévu à cet effet.

Attention, vous ne disposez que d'une seule tentative pour répondre à ces types de test de compréhension ! 

2. Des exercices de manipulation à réaliser depuis des pages web spécifiées (ex. : 4.3.1 et 5.1.2)
Vous pouvez effectuer autant de fois que vous le souhaitez ce type de test.
