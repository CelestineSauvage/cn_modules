(Q1/3)::[html]

Quels supports sont utilisés pour la transmission de données et qui peut donc servir pour une connexion internet ?

  • L'eau dans un tuyau.
  • L'électricité dans les câbles téléphonique.
  • L'électricité dans les câbles spécialisés.
  • La lumière dans les fibres optiques.
  • Les ondes électromagnétiques dans l'air.
  • L'électricité dans les câbles des prises électriques.

(Q2/3)::[html]

Qu'est-ce qui joue sur la vitesse d'une connexion ?

  • Le débit maximal admissible du lien qui me raccorde au réseau local.
  • Le nombre d'utilisateurs de ce lien.
  • Le nombre de messages qui passent par les mêmes routeurs que les miens.
  • Le nombre de requêtes arrivant sur la machine avec qui je désire échanger des messages.
  • L'âge de l'utilisateur.

Pour utiliser le wifi à Lille 3 je dois

  • Avoir un compte Lille 3 ou un compte dans une autre université.
  • Payer un abonnement.
  • Aller en cours.
